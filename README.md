# 🚦 交通路口信号灯配时模拟游戏

一个功能丰富的交通信号灯配时模拟工具，支持多种控制模式、公交优先、行人过街和真实数据导入。

## ✨ 核心功能

### 🎛️ 三种信号控制模式

1. **固定配时模式** - 传统的固定周期信号控制
   - 可调整信号周期长度（30-120秒）
   - 南北/东西向绿信比独立调节
   - 包含黄灯和全红清空相位

2. **感应式信号控制** - 基于实时车流量的智能控制
   - 车辆排队长度检测
   - 最小绿灯时间保障
   - 自适应延长绿灯时间
   - 配合公交优先效果更佳

3. **绿波带控制** - 干线协调优化
   - 可调绿波速度（20-80 km/h）
   - 带宽比例调节（10%-60%）
   - 最大化主干道通行效率

### 🚌 公交优先模式

- 公交车识别和单独检测
- 公交检测到后自动延长绿灯
- 公交车拥有更高的评分权重
- 可设置公交比例（0-30%）

### 🚶 行人过街系统

- 独立的行人信号灯
- 行人检测和过街逻辑
- 闪烁绿灯提醒功能
- 斑马线可视化
- 可设置行人流量

### 📊 数据导入与方案对比

- 支持导入真实路口JSON数据
- 方案自动按评分排序
- 多维度方案对比表格
- 本地存储持久化保存

## 🎮 评分机制

系统综合以下指标计算配时方案得分：

```
得分 = (通行量 × 10) + (公交通行 × 5) + (行人通行 × 2) - (平均等待 × 5) - (最大排队 × 2)
```

## 📁 数据格式

导入的真实路口数据需遵循以下JSON格式：

```json
{
  "intersectionName": "路口名称",
  "source": "数据来源",
  "cycleLength": 90,
  "nsGreenRatio": 0.55,
  "ewGreenRatio": 0.45,
  "nsFlowRate": 12,
  "ewFlowRate": 8,
  "busRatio": 0.15
}
```

## 🚀 快速开始

1. 在浏览器中打开 `index.html`
2. 选择信号控制模式
3. 调整各项参数
4. 点击"开始模拟"
5. 观察运行效果
6. 保存优秀配时方案

## 💡 使用建议

1. **早高峰场景** - 启用公交优先，提高主干道绿信比
2. **平峰时段** - 使用感应式控制，降低周期
3. **主干道协调** - 绿波模式+较高速度设置
4. **商业区路口** - 启用行人过街，设置合理行人流量
